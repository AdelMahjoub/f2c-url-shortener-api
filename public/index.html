Url shortener

Usage

Api endpoint: https://shy-save.glitch.me/api

Query string: ?url=[url to shorten]

Example

Request

https://s-url.glitch.me/api?url=https://gitter.im/FreeCodeCamp/FreeCodeCamp

Response

{
  "full": "https://gitter.im/FreeCodeCamp/FreeCodeCamp",
  "short": "shy-save.glitch.me/SypRsevHb"
}
